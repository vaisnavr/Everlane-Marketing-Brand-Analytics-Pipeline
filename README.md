# Everlane-Marketing-Brand-Analytics-Pipeline


## Overview
This project evaluates Everlaneâ€™s current brand positioning and performance using
secondary data, sentiment analysis, geographic profiling, and primary survey analytics.
The goal is to diagnose performance challenges and identify a viable strategic path forward.

## Data & Methods
- Google Trends competitive analysis
- Reddit and Trustpilot sentiment analysis
- ESRI Tapestry geographic profiling
- Primary consumer survey (factor analysis, segmentation, predictive modeling)

## Key Insights
- Strong product equity but weak operational trust
- Declining top-of-funnel interest relative to competitors
- Clear opportunity to blend quality leadership with modern, trend-forward design

## Strategic Recommendations
- Extend transparency to logistics and operations
- Redesign digital shopping and returns experience
- Shift brand messaging toward high-performance ethical fashion
- Focus on high-value urban customer segments

## Tools Used
- Python (pandas, sklearn, sentiment analysis)
- Excel / survey analytics
- ESRI demographic data
